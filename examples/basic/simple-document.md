# シンプルなドキュメント例

## 概要

これは基本的なMarkdownドキュメントの例です。見出し、段落、リスト、コードブロックなどの基本要素を含んでいます。

## 機能一覧

このツールは以下の機能を提供します：

- Markdownから PDF への変換
- Markdownから Word への変換
- 自動改ページ機能
- プロフェッショナルなスタイリング

## コード例

簡単なPythonコードの例：

```python
def hello_world():
    print("Hello, World!")
    return True

if __name__ == "__main__":
    hello_world()
```
